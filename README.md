# Subsidy Application GPT for SMEs (中小企業補助金申請GPT)

> “Streamline your subsidy applications and boost your chances of success.”

---

## 📌 Overview

**Subsidy Application GPT for SMEs (中小企業補助金申請GPT)** is a GPT-powered assistant designed to help Japanese small and medium-sized enterprises (SMEs) create application drafts for government subsidies, especially the “Monozukuri Subsidy” (ものづくり補助金).

It explains subsidy program details, highlights key success points, and generates clear drafts tailored for manufacturing and IT businesses.  
This tool **saves time, reduces stress, and increases the chance of a successful subsidy application.**

---

## ⚙️ Key Features

- ✅ Provides an overview of the Monozukuri Subsidy
- ✅ Explains critical points to improve approval chances
- ✅ Creates simple drafts tailored for manufacturing and IT industries
- ✅ Conducts additional Q&A if details are missing
- ✅ Outputs moderately formal Japanese suitable for official documents

---
